# BOGO Offer Page

## Overview
This is a pre-payment plan selection page for the BOGO (Buy One, Get One) offer. Users can choose from multiple plans, each with different discounts and options.

## Features
- **Multiple Plans**: Choose from 1, 2, or 3 unit plans with varying discounts.
- **Size and Color Options**: Select size and color for each unit.
- **Free Delivery**: All plans include free delivery.
- **Responsive Design**: Works on all devices.

## How to Use
1. Select a plan by clicking on the desired offer box.
2. Choose size and color options for each unit.
3. Review the total price and click "Add to Cart" to proceed.
